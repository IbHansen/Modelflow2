Jupyter Stuff
+++++++++++++++

This section contains different modules which are specific to the Jupyter interface. 
Jupyter allows interactive input and nice structured output. Some of the capabilities are present at 
several modules. It has been paramount to maintain backward compability. 

The **modeljupytermagic** module sets up different magic cell commands which can be handy. 

:graphviz: Draw Graphviz graph
:dataframe: Create Pandas Dataframe    
:latexflow: Create a modelflow modelinstance from latex script
